# Screenshots go here

Add one PNG per page (suggested naming in `../README.md`). The designer needs to see how each page renders with real data — code alone misses density, hierarchy, and the feel of an actually-used app.

Tips:

- Capture at desktop width (~1440px) — this is the primary form factor
- Use realistic data, not placeholder text
- Include both an "empty state" and a "populated state" screenshot for any page where they look meaningfully different (Search, Signals, Top by Signal Score, etc.)
- Don't worry about cropping perfectly; the designer just needs context
